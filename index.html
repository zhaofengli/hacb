HersfoldArbClerkBot Source Code
HersfoldArbClerkBot is coded in Java using the bot framework written by MER-C. The bot's source files are provided below. These files will be updated whenever the bot is, however this is not the bot's "live" code - it runs from a JAR compiled on my local computer.
  • Wiki.java - Basic framework for the bot to interface with the MediaWiki API
  • HersfoldArbClerkBot.java - Bot's main operational code
  • NewMessagesException.java - An Exception to be thrown when the bot receives new messages on its talk page
  • HACBModule.java - An abstract class containing generic methods needed to perform specific tasks
  • HACBEvidenceModule.java - An implementation of HACBModule containing the logic needed to perform the bot's evidence length enforcement task
